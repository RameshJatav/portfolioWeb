Node.js Intermediate & Advanced Interview Questions

1. How do you use Sequelize to join two tables?

In Sequelize, you can perform a join using the include option. For example, when retrieving a user, you can include their associated posts using:

const users = await User.findAll({
    include: {
        model: Post,
        attributes: ['title', 'content'],
    },
});

2. What are Sequelize hooks and how can they be used?

Sequelize hooks are lifecycle methods that allow you to run functions before or after certain events like creating, updating, or deleting records. They are helpful for validations or logging purposes. Example:

User.beforeCreate((user, options) => {
    user.name = user.name.toUpperCase(); // Capitalize name before creating
});

3. How do you handle pagination in Sequelize?

Pagination in Sequelize can be done by using the limit and offset options. Example:

const posts = await Post.findAll({
    limit: 10, // Number of posts per page
    offset: (page - 1) * 10, // Page number
});
